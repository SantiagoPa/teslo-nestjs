# Teslo API

1. clone project
2. install dependencies ```yarn install```
3. copy file ```.env.example``` and rename to ```.env``` - and fill fields
4. up db
```
docker compose up -d
```
5. up dev mode ```yarn start:dev``` 

6. Execute Seed (type: POST)
```
localhost:3000/api/seed
```
